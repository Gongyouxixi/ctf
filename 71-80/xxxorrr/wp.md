```
#include <stdio.h>
int main(){
    // 使用双引号初始化字符串
    char s1[] = "qasxcytgsasxcvrefghnrfghnjedfgbhn";
    for(int i=0;s1[i]!='\0';i++){
        s1[i]^=(2*i+65);
    }
    char s2[] = {
        0x56,0x4E,0x57,0x58,0x51,0x51,0x9,0x46,0x17,0x46,0x54,0x5A,0x59,0x59,0x1F,0x48,0x32,0x5B,0x6B,0x7C,0x75,0x6E,0x7E,0x6E,0x2F,0x77,0x4F,0x7A,0x71,0x43,0x2B,0x26,0x89,0xFE
    };
    for(int i=0;s1[i]!='\0';i++){
        s1[i]^=s2[i];
    }
    printf("%s\n",s1);
    return 0;
}
```
这个题挺有意思的，如果只看main追踪的话会少一个s1^=(2*i+65)的操作，只有追踪s1才能发现这个操作，勉强算难度3吧
